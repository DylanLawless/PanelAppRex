For case report validations, we selected the first five genetic diagnosis case reports using exome or genome sequencing from each of the journals of JACI (Immunology), Neurology, and any from PubMed in the last 12 months with free full text.
For the case report validations, we use the prompt: "Give me the medical phenotype terms from this text, then rank by most to least importance or relevance." followed by the original clinical case report without any content added after the genetic analysis layer from original reports. The search terms are added incrementally until no panels match.
